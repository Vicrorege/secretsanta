# Secret santa/Тайный санта
##### Русский
Тайный санта - традиционная рождественская игра, целью которой является анонимный обмен подарками в группе играющих людей. Участие в этой, как и в любой другой игре, является добровольным. По этой причине, к этой игре часто прибегают на работе или в больших семьях.

Этот код является генератором телеграм бота, который поможет вам сделать распределение обменивающихся людей

Для этого достаточно добавить ваш токен в config.py.
